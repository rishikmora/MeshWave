# Field Readiness Checklist

- Confirm firmware region plan matches local regulation.
- Confirm every relay has antenna attached before power.
- Validate BLE pairing with public key QR exchange.
- Send direct LoRa packet and verify ACK.
- Test two-hop relay path.
- Force node offline and verify route cache expires.
- Trigger emergency broadcast and verify elevated TTL.
- Run battery reserve test.
- Export diagnostics after field exercise.
